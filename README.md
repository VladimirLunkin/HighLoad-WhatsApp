# HighLoad-WhatsApp
Технопарк Mail.ru 3 семестр. Домашнее задание №1 по курсу "HighLoad".

## 1. Тема и целевая аудитория

**WhatsApp** - Cервис обмена мгновенными сообщениями.

### MVP
 - регистрация
 - чат (текстовые сообщения, голосовые сообщения, отправка файлов)
 - групповой чат
